## Project Overview
The Granite Shop Website is a comprehensive platform designed to showcase the products and services offered by a granite shop. The website features detailed product listings, an image gallery of past projects, customer testimonials, and an easy-to-use contact form. The goal is to provide potential customers with all the information they need to choose the perfect granite for their projects.

## Features
- **Responsive Design**: Mobile-friendly design to ensure usability on all devices.
- **Product Listings**: Detailed pages for each granite slab, including images and specifications.
- **Gallery**: Showcasing completed projects to inspire potential customers.
- **Contact Form**: Easy-to-use form for customer inquiries.
- **SEO Optimized**: Ensuring better visibility on search engines.
- **Blog Section**: Articles and tips related to granite products and their maintenance.
- **Social Media Integration**: Links to the shop's social media profiles.

## Technologies Used
- **HTML5 & CSS3**: For the structure and styling of the website.
- **JavaScript**: For interactive elements and enhanced user experience.

## Installation and Setup
### Steps
1. **Clone the repository**:
    ```sh
    git clone https://github.com/your-username/granite-shop-website.git
    ```

2. **Navigate to the project directory**:
    ```sh
    cd granite-shop-website
    ```
    
## Usage
- **Home Page**: View featured products and latest projects.
- **Product Listings**: Browse and filter granite products by various categories.
- **Gallery**: Explore the gallery for inspiration.
- **Contact Form**: Submit inquiries through the contact form.
- **Blog**: Read articles and tips about granite.
